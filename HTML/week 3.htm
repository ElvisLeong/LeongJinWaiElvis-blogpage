Hosting on GitHub

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.

So why would one use Github? There are several reasons,
Version Controll
  • Easy backtracking of the projects
  • Consistent backups
Project management
  • Helps doing work everywhere and not be tethered to 1 computer.
  • Determine timeframe and plan out your project
  • Resources
  • Easy collaboration
